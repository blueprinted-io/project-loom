Rules

Operational settings that control import governance.

Import behaviour

Controls the initial status assigned to records created via JSON or PDF import. Imported records are never set to confirmed regardless of this setting — confirmation remains a human-only trust boundary.

Features

Enable or disable major feature areas. Disabling a feature hides all related UI and blocks direct URL access. Existing records in the database are preserved and can be restored by re-enabling.
